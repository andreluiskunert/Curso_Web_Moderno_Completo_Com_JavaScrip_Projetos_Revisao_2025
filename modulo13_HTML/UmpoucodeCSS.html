Introdução ao CSS no HTML

O CSS é usado para dar estilo ao HTML. Ele pode ser aplicado de três formas principais:

  • Inline – dentro da própria tag HTML.
  • <style> – no cabeçalho da página.
  • Arquivo externo – recomendado, usando <link>.

Dois atributos importantes:

  • id → identificador único de um elemento.
  • class → permite aplicar estilos a vários elementos.

Um seletor CSS pode ser:

  • Elemento: h1 { }
  • Classe: .fundo-vermelho { }
  • ID: #texto-principal { }

Exemplo de CSS:

h1 {
  color: blue;
  font-size: 30px;
}

.fundo-vermelho {
  background-color: red;
  color: white;
}

O CSS permite organizar melhor o layout, separar conteúdo de estilo e reaproveitar formatações.
