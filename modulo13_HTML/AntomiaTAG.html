Resumo - Anatomia das Tags HTML

Introdução

O capítulo apresenta os conceitos fundamentais de HTML, mostrando que ele é uma linguagem de marcação de hipertexto. Também aborda a relação com CSS e JavaScript, que serão explorados em conjunto para enriquecer a aprendizagem.

Tags e Estrutura

As tags são a base do HTML. Elas podem ter corpo (com conteúdo interno e fechamento) ou não (como <meta>, <br>, <input>).

É possível aninhar várias tags, formando uma árvore hierárquica que organiza o documento. Algumas possuem parâmetros/atributos, que definem propriedades específicas.

Atributos e Personalização

Tags podem conter atributos padrões (como type em <input>) e também atributos personalizados, criados pelo desenvolvedor. Estes podem ser usados por frameworks ou scripts JavaScript para aplicar lógicas e estilos dinâmicos.

Flexibilidade do HTML

O navegador é flexível ao interpretar erros e tenta renderizar a página mesmo que haja falhas na escrita. Ainda assim, é recomendado validar o código em ferramentas adequadas para garantir qualidade.

Componentes e Evolução

Com HTML5 e frameworks como Vue ou React, é possível criar componentes personalizados, encapsulando HTML, CSS e JavaScript em novas tags, como um calendário ou formulário estilizado.

Conclusão

O objetivo do capítulo inicial é compreender a anatomia das tags, a estrutura de um documento HTML e as possibilidades de extensão e integração com CSS e JavaScript. Em seguida, serão criados os primeiros documentos HTML completos.
