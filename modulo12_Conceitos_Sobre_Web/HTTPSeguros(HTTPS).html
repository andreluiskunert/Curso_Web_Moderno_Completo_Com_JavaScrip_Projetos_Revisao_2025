Resumo sobre HTTPS

O que é HTTPS?

O HTTPS é a versão segura do protocolo HTTP. Ele utiliza criptografia para proteger os dados trocados entre o navegador e o servidor, garantindo maior segurança na navegação.

Como funciona?

  • O "S" vem de Secure (seguro).
  • A segurança é feita usando protocolos TLS (Transport Layer Security) ou SSL (Secure Sockets Layer).
  • Os dados são criptografados, dificultando ataques como o "homem do meio".
  • É necessário instalar um certificado digital no servidor.

Certificado x Protocolo

É importante diferenciar:

  • Protocolo: SSL ou TLS, responsáveis pela criptografia.
  • Certificado: arquivo emitido por uma Autoridade Certificadora (CA) que valida a identidade do site.

Autoridades Certificadoras

São entidades responsáveis por emitir e validar certificados digitais. Exemplo: Let's Encrypt, que oferece certificados gratuitos e confiáveis.

Por que usar HTTPS?

  • Protege dados sensíveis (senhas, informações pessoais, etc.).
  • Evita que terceiros interceptem a comunicação.
  • Passa mais confiança para os usuários.
  • É recomendado para qualquer aplicação moderna.

HTTPS = HTTP + TLS/SSL + Certificado Digital
